# 『高慢と偏見』翻訳統一方針書

## 1. 目的
本資料は、ジェーン・オースティンの『高慢と偏見』（Pride and Prejudice）における固有名詞、特殊用語、および表記の絶対的なルールを定義します。
当時のイギリスの階級社会や文化を正確に伝えるため、用語の統一は物語の説得力を維持する上で極めて重要です。

---

## 2. 固有名詞・特殊用語の統一ルール

### A. 人名・敬称
敬称は当時の社会的距離感を表現するため、原則としてカタカナ表記とします。

| 原文 | 日本語統一表記 | 備考 |
| :--- | :--- | :--- |
| Mr. Bennet | ミスター・ベネット / ベネット氏 | 文脈により「ベネット氏」も可とするが、会話文では「ミスター・ベネット」を推奨。 |
| Mrs. Bennet | ベネット夫人 / ミセス・ベネット | 地の文では「ベネット夫人」、会話文で呼びかける際は「ミセス・ベネット」。 |
| Miss Bennet | ミス・ベネット | 通常は長女ジェーンを指す。 |
| Elizabeth / Lizzy | エリザベス / リジー | 親しい間柄では「リジー」。 |
| Mr. Darcy / Fitzwilliam Darcy | ミスター・ダーシー / フィッツウィリアム・ダーシー | 地の文では「ダーシー」、会話では「ミスター・ダーシー」。 |
| Mr. Bingley / Charles Bingley | ミスター・ビングリー / チャールズ・ビングリー | 地の文では「ビングリー」。 |
| Lady Catherine de Bourgh | レディ・キャサリン・ド・バーグ / キャサリン令夫人 | 「レディ」は称号。会話では「レディ・キャサリン」で統一。 |

## 3. 記号・見出し・数字のルール（縦書き対応・厳守）

本書は縦書きのEPUBとして出版されるため、以下のルールを絶対に守ってください。

### A. 段落と字下げ
- **段落の字下げ禁止**: Markdown形式で出力するため、**段落の冒頭に全角スペースや空白を絶対に入れないでください**。字下げはEPUB化の際のCSSで制御します。

### B. 数字の表記
- **原則**: 本文中の数字は、すべて**漢数字**を使用します。
- **金額**: `10,000 pounds` → `一万ポンド`
- **年齢・時間・日付**: `twenty years old` → `二十歳`、`five o'clock` → `五時`
- **例外**: アラビア数字（1, 2, 3...）は使用しないでください。

### C. アルファベットの処理
- **1文字のアルファベット**: 原則として**全角**にしてください。
- **英単語**: 可能な限りカタカナ表記にしてください。（例：`piano` → `ピアノ`）

### D. 見出し
原文の章立て構造を維持しますが、数字は漢数字にします。
- 例: `## Chapter I.` → `## 第一章`
- 例: `## Chapter X.` → `## 第十章`

## 4. 文体・文末表現の統一

### A. 地の文
- **原則**: 「だ・である」調を基本とします。
- **トーン**: オースティン特有のアイロニー（皮肉）とユーモアを含んだ、洗練された流麗な文体。堅苦しすぎず、かといって軽薄にならない「モダン・クラシック・スタイル」を目指します。

### B. 会話文
- キャラクター設定（`proc-004-character.md`）で定義された各人物の口調を厳格に守ります。
- 階級や相手との関係性によって、言葉の丁寧さや距離感を調整してください。
- （例：エリザベスの快活な「〜だわ」「〜して？」、ダーシーの硬固な「〜である」「〜だろう」）
