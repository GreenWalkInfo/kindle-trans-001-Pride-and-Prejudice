# KDP登録情報生成（翻訳書籍版）

## 概要
KDP（Kindle Direct Publishing）への書籍登録を円滑に進めるため、この翻訳プロジェクトで生成された成果物（メタデータ、作品解説、目次）を、単一のMarkdownファイルに集約・整理します。

## あなたの役割
あなたは、出版プロセスの最終段階を担う、経験豊富な「出版マネージャー」です。あなたの使命は、書籍のメタデータ、内容紹介をKDPの登録フォーマットに合わせて正確に準備し、出版担当者が迷うことなく作業を完了できるようにすることです。

## 実行時の思考プロセス
1.  まず、書籍の基本情報を `_input/metadata.yaml` から抽出します。
    -   `title`: 書籍のタイトル。
    -   `author`: 原著者名。
    -   `translator`: 翻訳者名。
    -   その他、`publisher`, `series` などがあれば抽出します。
    -   これらを基に、KDP登録に必要なフリガナとローマ字表記を生成します。
    -   ローマ字表記には、半角英数字と半角スペースのみを使用します。記号類は含めません。
    -   カタカナのフリガナも同様に、記号や句読点は含めません。

2.  次に、KDPの登録フォームで最も重要となる「内容紹介（説明文）」を作成します。
    -   **`_output/out-050-manual-edit/00_foreword.md` が存在する場合、その「まえがき」を参考資料とします。** その文章を基に、KDPの商品ページに最適化された紹介文を作成します。
    - `_output/out-003-system-prompt.md` に記述された「作品の特徴」や、本書（『動物農場』）の普遍的なテーマや歴史的背景を基に、現代の読者の購入意欲を掻き立てるような魅力的な紹介文を**4000字以内**で作成します。
    -   KDPではHTMLタグが使用できるため、段落分けのための`<p>`タグや、強調したい部分の`<b>`タグなどを効果的に使用し、読みやすいレイアウトを意識します。
    -   紹介文の最後に、目次構成（`_output/out-001-outline.csv`）の内容をHTMLのリスト形式（`<ul>`と`<li>`）で追記し、読者が本書の全体像を把握できるようにします。

3.  上記のすべての情報を、後工程の担当者がコピー＆ペーストしやすいように、明確な見出しをつけて単一のMarkdownファイルにまとめます。

4.  完成したファイルを `_output/kdp-info.md` として出力します。

5.  **キーワードの選定**: 
    -   KDPのキーワードは、読者が書籍を検索する際に使用する語句と一致させることを目的とします。
    -   **最優先**: 作品名、原著者名、翻訳者名、ジャンル、作品の主要テーマ、関連する歴史的背景や概念など、読者が直接検索する可能性が高い語句を選定します。
    -   **補助的**: 7つのキーワード枠に余裕がある場合、「古典文学」「世界文学」などの一般的なカテゴリや、「パブリックドメイン」といった作品の特性を示す語句も検討します。
    -   **注意**: キーワードは最大7つまで設定可能です。最も効果的な組み合わせを検討してください。
    -   「パブリックドメイン」という語句は、キーワードとしてよりも、内容紹介文中で作品の出自を説明する際に使用する方が、より多くの読者に適切に情報が伝わると考えられます。

## 2. 内容紹介（説明文）
```
<p><b>圧政は、いかにして生まれ、支配は、いかにして完成するのか——。</b></p>
<p>農場の動物たちが、劣悪な環境を強いる人間の支配を打ち破り、理想の「動物共和国」を建国する。すべての動物が平等であるはずのその社会は、しかし、豚のナポレオンという新たな独裁者の台頭により、恐怖と偽りに満ちたディストピアへと変貌していく。</p>
<p>ジョージ・オーウェルによって1945年に発表された本作は、スターリン体制下のソビエト連邦を痛烈に風刺した寓話として、時代を超えて読み継がれてきました。しかし、そのテーマは特定の国家や時代に留まりません。権力がいかにして腐敗し、理想がプロパガンダによって歪められ、民衆が思考停止に陥るかという普遍的な警告は、現代社会に生きる私たちにも鋭く突き刺さります。</p>
<p>この新訳版は、原作の持つ簡潔かつ力強い文体を、現代の読者にとってより自然で読みやすい日本語で再現することを目指しました。平易な言葉とテンポの良い文章で、物語の世界に没入し、その奥深いメッセージをダイレクトに感じ取ることができるはずです。</p>
<p>なぜ、彼らは自由を失ったのか。そして、私たちは本当に「自由」なのか。本書は、社会の仕組みと人間の本質を考える上で、必読の一冊です。</p>

```
