# proc-007-annotation: 注釈付与ルールの作成

## 1. 目的
`out-011-translated`等の翻訳フェーズにおいて、Gemini 3.1 Proに「Markdownの脚注形式」で注釈を同時に生成させるためのプロンプト部品（ルール書）を作成する。

## 2. 出力ファイル
- `_output/out-007-annotation.md`

## 3. 記載すべき内容
1. **注釈の書式**: `[^1]`（インライン）と `[^1]: 注釈内容`（ファイル末尾）のMarkdown脚注形式を厳守する指示。
2. **対象基準**: 18世紀末〜19世紀初頭のイギリス社会の風俗、階級制度、地名、法律（限嗣相続など）といった、現代の読者に馴染みのない文化的背景の説明に絞る。
3. **トーン＆マナー**: 学術的な長文解説ではなく、「物語の理解を助ける」「キャラ文芸のテンポを崩さない」簡潔なエンタメ志向の注釈にするよう指示する。
4. **処理の統合**: 翻訳セクションを出力した直後に、セクションの末尾にまとめて注釈本文を出力することを指示する。

---
このファイルは構成指示書です。実際のルール内容は以下のコマンドで `_output/out-007-annotation.md` に出力させてください。

## 実行タスク

Geminiに以下のプロンプトを投げて、出力結果を `_output/out-007-annotation.md` に保存してください。

```markdown
以下の条件に従って、ジェイン・オースティンの『高慢と偏見』の翻訳時における「注釈付与ルール」を作成してください。

1. **目的**: 翻訳テキストと同時に、Markdown形式の脚注として注釈を生成するため。
2. **形式**: インラインで `[^1]` のように連番を振り、ファイル末尾に `[^1]: 注釈内容` として解説を出力すること。
3. **付与基準**:
    *   18世紀末〜19世紀初頭のイギリス社会・風俗（馬車、舞踏会の習慣など）。
    *   階級制度（ジェントリー階級の仕組み、称号など）。
    *   法律や制度（限嗣相続など、物語の理解に直結するもの）。
    *   当時の地理や実在の場所。
4. **トーン＆マナー**: 学術的になりすぎず、物語のテンポを損なわない簡潔でわかりやすい解説とすること。
5. **出力位置**: 翻訳テキストの**最後尾**に必ず脚注リストをまとめて出力すること。

出力はそのままルールファイルとして利用できるよう、説明や前置きを省き、ルール本文のみとしてください。
```
