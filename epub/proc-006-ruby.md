# proc-006-ruby: ルビ付与ルールの作成

## 1. 目的
`out-011-translated`等の翻訳フェーズにおいて、Gemini 3.1 Proに「青空文庫形式のルビ」を同時に生成させるためのプロンプト部品（ルール書）を作成する。

## 2. 出力ファイル
- `_output/out-006-ruby.md`

## 3. 記載すべき内容
1. **ルビの書式**: `|親文字《ルビ》` の形式（青空文庫形式）を厳守する指示。
2. **対象基準**: 難読漢字、ジェントリー階級の特有の用語、固有名詞（当て字の場合）などに絞る。過剰なルビ（小学生レベルの漢字など）は避ける。
3. **処理の統合**: 翻訳テキストの出力と同時に、インラインでこのマークアップを埋め込むことを指示する。

---
このファイルは構成指示書です。実際のルール内容は以下のコマンドで `_output/out-006-ruby.md` に出力させてください。

## 実行タスク

Geminiに以下のプロンプトを投げて、出力結果を `_output/out-006-ruby.md` に保存してください。

```markdown
以下の条件に従って、ジェイン・オースティンの『高慢と偏見』の翻訳時における「ルビ付与ルール」を作成してください。

1. **目的**: 翻訳テキストにインラインで青空文庫形式のルビを付与するため。
2. **形式**: `|親文字《ルビ》` の形式を厳守すること。
3. **付与基準**:
    *   18世紀末〜19世紀初頭のイギリス社会（特にジェントリー階級）の専門用語や難読語（例: 従男爵、限嗣相続など）。
    *   一般的な読者が読みにくいと感じる可能性のある難読漢字。
    *   一般的な漢字（私、彼、行く など）にはルビを振らないこと。
4. **トーン**: 翻訳時にAIが理解しやすい、明確で簡潔な指示書の形式にすること。

出力はそのままルールファイルとして利用できるよう、説明や前置きを省き、ルール本文のみとしてください。
```
